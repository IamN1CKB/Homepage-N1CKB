CSS grid

Per sviluppare meglio uno schema in 2 dimensioni possiamo usare il CSS grid (griglia).
Per strutturare un css grid bisogna sempre creare un tag div con un attributo class o id

Per iniziare con la griglia, nel foglio di stile dobbiamo applicare nel grid-containerla propretà "display: grid;"
Inoltre inseriamo anche le proprietà border, background-color, color, margin e tex-align (center) per capire meglio come sono disposti gli elementi

Possiamo inoltre dare una altezza (height) e larghezza (width) alla griglia, in modo da poter usare i valori in percentuale

Per organizzare i grid items in colonne useremo la proprietà "grid-template-columns" nel tag del grid container
Setttiamo la grandezza di ogni colonna mettendo tot valori (della proprietà template...) quante sono le colonne e ogni valore corrisponde a alla grandezza di una colonna

Ad esempio con la proprietà "grid-template-columns: 50px 50px 120px;" ci saranno 3 colonne. La prima colonna sarà larga 50px, la seconda colonna sempre 50px, e la terza colonna 120px.
Nel caso di un numero di grid items maggiore del numero di colonne, gli elementi extra vanno a capo

RICRODA: la grandezza delle colonne e delle righe si setta sul parent element (in questo caso il grid container)

  3. il valore auto (tra poco vediamo)
  4. valori in fr (dopo vediamo cos'è)

grid gaps

di default, i grid items non hanno spazi tra di loro che li dividono. di solito ci arrangiavamo con la proprietà dei flex item "margin: 2px;"
ora utilizziamo una nuova proprietà (con una nuova classe di grid-item) in cui usiamo le varie funzione gap (contratte o meno).
dunque la differenza tra le funzioni gap e la funzione margin è che con margin possiamo modificare gli spazi tra un elemento e l'estreno, e invece con la funzione gap modifichiamo gli spazi tra i grid items
ad esempio se tutti i grid item hanno i margini di 5px, lo spazio tra gli elementi sarà 10px, siccome è la somma dei margini di uno e di un altro grid item
invece con le funzioni gap si stabilisce tutto lo spazio tra i grid item, ossia con il valore del gap uguale a 5px, vuol dire che tra i grid items ci sarà 5 px di spazio
RICORDA: LE FUNZIONI GAP SI INSERISCONO NEL GRID CONTAINER, INVECE I MARGIN NEI GRID items

Grandezza dei grid items

i grid items possono occupare x numero di colonne e y numero di righe, definendo così le loro dimensioni
per far ciò ci sono 2 metodi:
  1. usare le proprietà "grid-column" e "grid-row"
  2. creare aree con i nomi
vediamo il primo caso.
ci sono le proprietà "grid-column" e "grid-row" che stabiliscono di quante righe o colonne un elemento si deve estendere
queste proprietà vanno INSERITE NEI GRID ITEMS (NO GRID-CONTAINER). I valori di queste proprietà sono formate dalla parola span (intervallo) e un numero.
esempio nel caso del primo grid item e del secondo e ventesimo:

.grid-item-1 {
grid-column: span 3;
}

.grid-item-2 {
grid-row: span 2;
}

.grid-item-20 {
grid-column: span 2;
grid-row: span 4;
}

Il secondo modo ci permette di dare dei nomi ad ogni grid items e posizionarlo in base al loro nome.
per prima cosa dobbiamo dare a tutti i grid items un attributo class diverso
poi nel foglio di stile aggiungiamo a ogni grid item la proprietà "grid-area", il cui valore segnerà un nome a quel grid item
poi dobbiamo aggiungere nel grid container la proprietà "grid-template-areas"
i valori di grid-template-areas devono essere i valori di grid-area dei vari grid items, scritti nell'ordine in cui verrano visti nella pagina.
in questo modo potrebbe pure non servire la proprità grid-template

ora stabiliamo alcune regole nelle quali devono essere scritti i valori di grid-template-areas

  • se vogliamo far estendere un grid item in più caselle, ripetiamo il suo valore grid-area
  • se vogliamo lasciare degli spazi vuoti scriviamo . tra le grid area che vogliamo lasciare fuori
  • se vogliamo diprorre gli elementi in righe diverse possiamo scrivere più valori in più ""
  • per comodità nel foglio di stile scriviamo i valori tra le "" in linee di codice successivi
  • tutte le linee di codice devono avare lo stesso numero di colonne (inserisci . dove non vuoi nulla), altrimenti errore
  • non ci devono essere valori ripetuti se non sono adiacenti (si trovano vicino verticalmente o orizzontalmente), altrimenti errore
  • i grid items possono estendersi o solo verticalmente o solo orizzontalmente (non figure ad L o simili), altrimenti errore
  • se ci sono dei grid items che non hanno avuto la proprietà grid-area, si inseriscono nei primi spazi vuoti (quelli col il . )
    • se finiscono gli spazi voti si posizionano in una nuova riga
  • se ci sono grid-item con il grid-area ma che non sono stati inseriti nel grid-template-areas, sballano il valore della tabella, quindi errore

Malattie sessualmente trasmittibili

Una malattia trasmissibile sessualmente (nota anche con l'acronimo MTS) o infezione sessualmente trasmissibile (IST), o in lingua inglese sexually transmitted disease (STD), in passato malattia venerea, nella medicina, indica una malattia infettiva che si trasmette principalmente per contagio diretto in occasione di attività sessuali, inclusi i rapporti orali e anali. La maggior parte delle MTS inizialmente non presenta sintomi e ciò si traduce in un maggior rischio di trasmetterle ad altre persone. I sintomi e i segni della malattia possono includere perdite dalla vagina, dal pene, dolore pelvico, ulcere intorno agli organi genitali o su di essi. Un'infezione sessualmente trasmessa acquisita prima o durante una gravidanza può causare danni al nascituro. Alcune malattie sessualmente trasmissibili possono causare problemi di sterilità

ALIGN GRID ITEMS

ROW AXIS
i grid items possono essere posizionati in varie parti della row axis (ossia la linea immaginari orizzontale) della casella che occupa il grid item. la proprietà che si occupa di ciò si chiama justify-items, e si inserisce nelle regole del grid container

    i valori di justify-itmes sono:
  • stretch: valore di default, che fa espandere il grid items in tutta la casella lungo il row axis.
  • start: che fa posizionare il grid item solo nella parte a sinistra della casella (parte iniziale della casella) lungo il row axis, occupando solo lo spazio minimo necessario
  • end: stessa cosa di start ma a destra (parte finale della casella)
  • center: fa posizionare il grid item solo al centro della casella lungo il row axis occupando meno spazio possibile

COLUMN AXIS
invece per posizionare i grid items lungo la column axis si usa la proprietà align-items, sempre inserita nel container

    i valori di align-items sono:
  • stretch: valore di default, che fa espandere il grid items in tutta la casella lungo il column axis.
  • start: che fa posizionare il grid item solo nella parte a sinistra della casella (parte iniziale della casella) lungo il column axis, occupando solo lo spazio minimo necessario
  • end: stessa cosa di start ma a destra (parte finale della casella)
  • center: fa posizionare il grid item solo al centro della casella lungo il column axis occupando meno spazio possibile
esempi:
justify-items: stretch; (default value)

USING FLEXBOX AND GRID RULE

per creare una "responsive" web page, ossia una pagina web adattiva a ogni tipo di schermo, possiamo combinare il flexbox con le grid rules
le grid CSS sono ottime per combinare un layout in 2 dimensioni, con righe e colonne.
Nonostante ciò, i CSS grid sono limitati per disporre gli elementi in un grid item, per cui si usano le regole del flexbox.
infatti se usiamo proprietà come justify-content o justify-items, noi muoviamo ogni elemento nella direzione selezionata, e non solo gli elementi del grid item voluto.
